
# Aplikasi Peminjaman Lengkap

Jalankan (lokal):

1. Install dependencies:
   ```
   npm install
   ```
2. Seed database (membuats data.db):
   ```
   python3 seed_db.py
   ```
3. Jalankan server:
   ```
   node server.js
   ```
4. Buka http://localhost:3000

Repositori ini siap di-push ke GitHub.
